# Objects

The `objects` controller allow you to direct manipulate each object on the canvas.

## Add

Objects can be added using `add` method.

- options: Object to be added

```ts
editor.objects.add(options)
```

Example: User wants to add a `StaticText` object.

```ts
const options = {
  type: "StaticText",
  width: 320,
  metadata: {
    text: "Hello Layehub",
    fontSize: 52
  }
}
editor.objects.add(options)
```


## Update

You can update selected object by:

- options: Properties being updated for the object.
- id(optional): Object target id. If not present, selected object will be updated.

```ts
editor.objects.update(options, id)
```

Example: User wants to update a `StaticText` object.

```ts
const options = {
  fontSize: 42,
  width: 240,
  textAlign:"center"
}
editor.objects.update(options)
```


## Selection

Objects can be selected/deselected programmatically using the API. Mouse events can also select/ deselect objects.

- id(optional): Object target id. If not preset, all objects will be selected

```ts
editor.objects.select(id)
```

In the same way, objects can also be deselected using:

```ts
editor.objects.deselect()
```

## Move

Objects can be moved by either dragging using mouse or programmatically using the API. It uses relative values.

- direction: `top` | `left`
- value: number
- id(optional): Object taget id. If not provided, will be applied to current selection.

```ts
editor.objects.move(direction, value, id)
```

Example: User wants to move object to the left.

```ts
editor.objects.move("left", 100)
```


## Position

Objects can be positioned by either dragging using mouse or programmatically using the API, It uses absolute values.

- direction: `top` | `left`
- value: number
- id(optional): string

```ts
editor.objects.position(position, value, id)
```

## Resizing objects

Objects can be resized by ether using pointer or programmatically using the API.

- direction: `width` | `height`
- value: number
- id(optional): string

```ts
editor.objects.resize(dimension, value, id)
```

## Scaling objects

Objects can be scaled by either using pointer or programmatically using the API.

```ts
editor.objects.scale(type, id)
```

## Layer

Objects layer position can be updated programmatically using the API.

- layer: `backwards` | `toBack` |`forwards` | `toFront`
- id(optional): string

```ts
editor.objects.layer(layer, id)
```

## Align

Objects can be aligned using the API

- alignment: `left` | `center`  | `right` | `top` | `middle` | `bottom`
- id(optional): string

```ts
editor.objects.align(alignment, id)
```

## Syntax Highlighting

Automatic syntax highlighting:

````markdown
```js
console.log('hello, world')
```
````

Renders:

```js

```

You can also add the `highlight=<line|range>` modifier to highlight specific lines:

````markdown

import useSWR from 'swr'

function Profile() {

  if (error) return <div>failed to load</div>
  if (!data) return <div>loading...</div>
  return <div>hello {data.name}!</div>
}
````
