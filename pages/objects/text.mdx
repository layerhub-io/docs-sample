# Text

Customizable text object. Text objects can be either static or dynamic. Dynamic text can accept variables.

```ts
interface Text { 
    id: string
    name: string
    top: number
    left: number
    width: number
    height:number
    opacity: number
    scaleX: number
    scaleY: number
    stroke: string
    strokeWidth: number
    angle: number
    flipX: boolean
    flipY: boolean
    shadow: Shadow
    visible: boolean
    metadata: {
        charSpacing: number
        fill: number
        fontFamily: string
        fontSize: number
        text: string
    }
}
```
