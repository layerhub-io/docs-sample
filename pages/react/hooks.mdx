# Hooks

## useEditor

Main hook which returns an instance of the editor. It gives you access to all features.

```tsx
import React from 'react'
import { useEditor } from '@layerhub-io/react'

function Sidebar() {
  const editor = useEditor()

  const addObject = () => {
    if (editor) {
      editor.objects.add(options)
    }
  }
  return <Box></Box>
}
```

## useZoomRatio

Hook which returns current zoomRatio.

```tsx
import React from 'react'
import { useZoomRatio } from '@layerhub-io/react'

function Sidebar() {
  const zoomRatio = useZoomRatio()
  return <Box>{zoomRatio}</Box>
}
```
